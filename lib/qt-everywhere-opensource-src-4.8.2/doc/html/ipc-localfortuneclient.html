Local Fortune Client Example

Files:

  • ipc/localfortuneclient/client.cpp
  • ipc/localfortuneclient/client.h
  • ipc/localfortuneclient/main.cpp
  • ipc/localfortuneclient/localfortuneclient.pro

The Local Fortune Client example shows how to create a client for a simple local service using QLocalSocket.

The example is intended to be run alongside the Local Fortune Server example.
