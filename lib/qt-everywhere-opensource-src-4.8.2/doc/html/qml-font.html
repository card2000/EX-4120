QML Basic Type: font

A font type has the properties of a QFont. The properties are:

  • string font.family
  • bool font.bold
  • bool font.italic
  • bool font.underline
  • real font.pointSize
  • int font.pixelSize

Example:

 Text { font.family: "Helvetica"; font.pointSize: 13; font.bold: true }

See also QML Basic Types.
