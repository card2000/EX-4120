Graphics View Examples

Qt is provided with a comprehensive canvas through the GraphicsView classes.

These examples demonstrate the fundamental aspects of canvas programming with Qt.

  • Colliding Mice*
  • Diagram Scene*
  • Drag and Drop Robot*
  • Elastic Nodes*
  • Pad Navigator*
  • Ported Asteroids
  • Ported Canvas

These examples show the use of graphics widgets and layouts.

  • Anchor Layout
  • Flow Layout
  • Simple Anchor Layout
  • Weather Anchor Layout
  • Basic Graphics Layouts

Some examples demonstrate the use of graphics effects with canvas items.

  • Blur Picker Effect
  • Fade Message Effect
  • Lighting Effect

Examples marked with an asterisk (*) are fully documented.
