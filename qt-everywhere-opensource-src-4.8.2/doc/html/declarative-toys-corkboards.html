Toys: Corkboards Example

Files:

  • declarative/toys/corkboards/corkboards.svg
  • declarative/toys/corkboards/qml/corkboards/Day.qml

This example presents a flickable set of interactive corkboards. It is created through a combination of elements like ListModel, Repeater and TextEdit together with rotation and scaling transforms, animation and mouse interaction.
