QSocketNotifier::QSocketNotifier ( int socket, Type type, QObject * parent, const char * name )

Use the QSocketNotifier() constructor combined with the QObject::setObjectName() function instead.

For example, if you have code like

 QSocketNotifier *notifier = new QSocketNotifier(socket, type, parent, name);

you can rewrite it as

 QSocketNotifier *notifier = new QSocketNotifier(socket, type, parent);
 notifier->setObjectName(name);
