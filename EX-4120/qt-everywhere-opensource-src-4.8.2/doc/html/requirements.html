General Qt Requirements

This page describes the specific requirements of libraries and components on which Qt depends. For information about installing Qt, see the Installation page.

For information about the platforms that Qt supports, see the Supported Platforms page.

OpenSSL (version 0.9.7 or later)

Support for Secure Sockets Layer (SSL) communication is provided by the OpenSSL Toolkit, which must be obtained separately. More information about enabling SSL support can be found in the Secure Sockets Layer (SSL) Classes document.
